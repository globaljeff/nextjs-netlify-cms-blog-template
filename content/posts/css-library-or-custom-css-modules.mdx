![website design](/images/byer-comp.jpg "Choosing between a css library or writing custom css modules")

First lets identify the pros and cons of using a css library. In a global sense, a css library will inlcude a ton of modules that some projects will never use. Ideally I want the css library to only use the components that are being used on the page and strip out the rest. This will increase performance and keep your site experience optimized for performance. 

The next thing I want to consider is third party dependencies. This will reduce maintenance while increasing reliability.
